## 接口列表

### 1.编辑精选排序接口

**接口** : /goods/querySelection.do

**请求方式**  get  

**参数** : 

``` 
不用传参
```

**返回** :
json

``` 

{
  "status": {
  "code": 1001,   1004    是错误
  "msg": "请求成功"
  }
}	
```
