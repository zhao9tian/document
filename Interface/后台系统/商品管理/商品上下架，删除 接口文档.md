## 商品上下架(B端或者C端)

### 接口名称 /goods/UpOrDown.do

#### 请求方式 :  get

#### 请求参数

```
{
  goodsId : 10002 ,					商品Id
  goodsSource : 10 , 20				商品来源  10:C    20:B
}
```

#### 返回

```
{
  status :{
    code : 1001 ,				1004 请求失败
    msg : "请求成功"
  }
}
```
