## 接口列表

### 1.获取专题排序

**接口** :     /goods/queryThemeSort.do

**请求方式**  ：get

**参数** : 

``` 
不用传参
```

**返回** :
json

``` 
	{
        "status": {
                    "code": 1001,   1004    是错误
                    "msg": "请求成功"
                    },
        "data":{ 
        		themeSort : [11,22,33]
        }
        	
    }	
```

#### 2.保存专题排序

##### 接口  :  /goods/addThemeSort.do

##### 请求方式  :  post

##### 参数 

```
{
  themeSort : [1,2,3]
}
```

##### 返回

```
{
        "status": {
                    "code": 1001,   1004    是错误
                    "msg": "请求成功"
                    }
 }	
```
